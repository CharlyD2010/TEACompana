# **App Name**: TEACompaña

## Core Features:

- Secure User & Child Profile Management: Enables user registration, login with email/password, and role-based access for parents/teachers. Adults can create, edit, and select individual child profiles, capturing details like name, TEA level, interests, and medical notes. Authentication and data storage are managed securely using Firebase.
- AI-Powered Initial Assessment & Learning Plan Generator: Facilitates a guided initial assessment across areas like emotions, communication, and motor skills for each child. Utilizing a generative AI tool, the app then creates a personalized learning plan based on assessment results, outlining prioritized areas, weekly objectives, recommended activities, and suggestions for both parents and teachers.
- Interactive Educational Game Module: Provides a selection of engaging, visually-rich games tailored for children with TEA, focusing on themes such as emotion recognition, colors & shapes, object counting, and daily routines. Games track individual performance including accuracy, time, and scores.
- Personalized Progress & Reward Tracking: Monitors and displays each child's game progress, completed activities, earned stars, and accumulated points. An automated reward system celebrates achievements like 'First activity completed' or 'Level X Unlocked' through visual badges.
- Comprehensive Reporting & Insights: Generates detailed weekly and monthly reports for parents and teachers, presenting insights into a child's learning progress, activity performance, areas of strength, and specific areas for improvement, alongside actionable recommendations.
- Supervised Kid Mode & Activity Selection: Offers a simplified, intuitive 'Kid Mode' interface allowing children to easily access and engage with recommended activities and games under adult supervision. This mode ensures age-appropriate content display and prevents unsupervised navigation.
- Robust Backend Integration (Firebase): Establishes a secure and scalable backend using Firebase, storing all essential application data, including user profiles, child data, assessment results, learning plans, game sessions, progress metrics, and rewards. This ensures data persistence and synchronization across devices.

## Style Guidelines:

- Primary pastel blue: `#86ACF5` (HSL: 220, 60%, 70%) to create a calm and supportive visual foundation for the application.
- Background color: `#F5F7FA` (HSL: 220, 15%, 97%) chosen as a warm, very light off-white or subtle grey-blue to provide a gentle and inviting base.
- Mint green accent: `#ADF0CA` (HSL: 150, 40%, 75%) utilized specifically for indicators of progress and positive feedback within the interface.
- Soft yellow accent: `#F5EF86` (HSL: 50, 70%, 85%) designated for highlighting elements like stars, achievements, and other reward indicators.
- Soft coral accent: `#F5AE97` (HSL: 10, 70%, 75%) applied to actions such as 'exit' or 'cancel,' ensuring clear and gentle differentiation for these interactions.
- Pastel lilac accent: `#D1ADF0` (HSL: 270, 40%, 75%) used for secondary elements and complementary details to add visual interest and a soft touch.
- Headlines and body text: 'PT Sans' (humanist sans-serif) will be used for its modern, highly readable, and inherently warm character, making it suitable for both the professional tone of adult sections and the friendly presentation for children.
- Large, clear, and engaging icons paired with custom illustrations will be extensively used throughout the children's section to enhance visual comprehension, appeal, and overall user engagement.
- The layout will feature prominently rounded cards, subtle and soft shadows, and generous spacing between elements to cultivate an inviting and easy-to-navigate experience. Interactive elements in child-focused sections will be notably large for ease of use.
- Subtle animations will be integrated for positive reinforcement (e.g., successful interactions, reward unlocks), smooth screen transitions, and interactive game components, enhancing user experience without creating overstimulation.
